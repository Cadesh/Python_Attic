This program calculates Precision and Recall for files with the folloing format:

video name and a sequence of booleans (0,1) representing the different categories.

The program used the testBooleanGounrdTruth.csv generated in code 28.1
and the unWeighted votes from inception with KNN10 genrated in code 28.5 and then 
turned to 28.1 to generate the booleans.
